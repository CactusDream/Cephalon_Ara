# 🌌 Cephalon Ara (세팔론 아라) - Beta v0.1.0

**세팔론 아라(Cephalon Ara)**는 한국 워프레임 플레이어를 위한 **실시간 정보 및 유틸리티 도구**입니다.
현재 **오픈 베타(Open Beta)** 테스트 중이며, 여러분의 피드백을 바탕으로 성장하고 있습니다.

---

## 주요 기능

현재 베타 버전에서는 다음 기능들을 즉시 사용할 수 있습니다.

### 1. 🌍 실시간 월드 정보
* **아이돌론 평원/오브 협곡/캠비온 퇴적지**의 낮/밤 사이클을 한눈에 확인하세요.
* 남은 시간을 실시간 타이머로 보여줍니다.

### 2. ⚔️ 미션 정보
* **중재:** 현재 중재 미션의 타입과 이후 중재 테이블을 확인하세요.
* **균열:** 성유물 등급별 균열 미션을 빠르게 찾으세요.

### 3. 🟣 리벤 도구
* **리벤 계산기:** 무기의 이름을 입력하여 옵션의 최저~최고치를 계산합니다.
* **리벤 스캐너 (Beta):** 리벤 순환 화면에서 화면 인식(F8)을 통해 리벤 모드 정보를 자동으로 읽어옵니다. *(현재 실험적 기능입니다)*

### 4. 🔔 알림
* 원하는 미션이나 보상이 떴을 때 알림을 받아보세요.

## 설치 및 실행 방법

1.  이 저장소의 우측 **[Releases]** 탭 또는 [여기(다운로드 링크)](https://github.com/CactusDream/Cephalon-Ara/releases)를 클릭합니다.
2.  최신 버전의 **`CephalonAra.exe`** 파일을 다운로드합니다.
3.  파일을 실행하여 설치를 진행합니다.

> **⚠️ 실행 시 'Windows의 PC 보호' 경고창이 뜨나요?**
> 개인 개발자가 만든 프로그램이라 발생하는 정상적인 현상입니다.
> **[추가 정보]** 클릭 후 **[실행]** 버튼을 누르시면 정상적으로 실행됩니다.

## ⚠️ 베타 서비스 안내 및 면책 조항

본 프로그램은 현재 개발 중인 **베타 버전**입니다.
* **데이터 정확성:** 제공되는 정보는 Warframe API를 기반으로 하지만, 실제 게임과 미세한 차이가 있을 수 있습니다.
* **책임의 한계:** 본 프로그램을 사용하여 발생한 게임 내 손실이나 데이터 오류에 대해 개발자는 법적 책임을 지지 않습니다.
* **버그 제보:** 사용 중 오류를 발견하시면 디스코드, 메일로 보내주세요. 빠른 수정에 큰 도움이 됩니다!
